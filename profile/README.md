# Geofast Games

Mobile game development studio.

## Product

**Geofast: Battle of Nations** - Turn-based strategy game for iOS and Android.

## Technology

- Game Engine: Godot 4.x (GDScript)
- Backend: Nakama (Go)
- Platforms: iOS, Android
- Services: Firebase, AdMob

## Open Source

Contributions to the Godot ecosystem:

- [SwiftGodotIosPlugins](https://github.com/geofast-games/SwiftGodotIosPlugins) - iOS plugins for Godot
- [godot_oauth_plugin](https://github.com/geofast-games/godot_oauth_plugin) - OAuth using Google's Credentials Manager  
- [APNsGodotPlugin](https://github.com/geofast-games/APNsGodotPlugin) - Apple Push Notification plugin
